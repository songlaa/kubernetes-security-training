Labs

The purpose of these labs is to convey Kubernetes security by providing hands-on tasks for people. Kubernetes (K8s) will allow you to deploy and deliver your software packaged as containers in an easy, straightforward way.

Goals of these labs:

  • Help you get started with this modern technology in a secure fashion
  • Explain the basic concepts to you
  • Show you how to deploy your first applications on Kubernetes
  • Show you how to secure your applications on Kubernetes
